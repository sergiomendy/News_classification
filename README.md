## Prérequis
Pour utiliser à bien ce depôt, veillez installer les bibliothèques listées dans le fichier requirements.txt ou faites seulement:
```
pip install -r requirements.txt
```


## Fichiers du dépôt

1. demo.py:
    - Description: Ce fichier est destiné à être utilisé pour la démonstration avec Gradio. 
    - Utilisation: Pour exécuter la démonstration avec Gradio, utilisez la commande suivante :
        ```bash
        python demo.py
        ```
2. main.py:
    - Description: Ce fichier est destiné à être utilisé pour la démonstration avec FastAPI. 
    - Utilisation: Pour exécuter la démonstration avec FastAPI, utilisez la commande suivante :
        ```bash
        uvicorn main:app --reload
        ```    
3. News_classification_using_bert.ipynb:
    - Description : Ce fichier Jupyter Notebook présente les différentes étapes de l'entraînement du modèle.
    - Utilisation : Vous pouvez ouvrir le notebook et suivre les étapes pour l'entraînement du modèle.
